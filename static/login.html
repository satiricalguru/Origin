Origin is a self-hosted platform running locally on your machine.

If you are the administrator:

  • You can reset your password or configure a new account by deleting or editing the database file at data/auth.json in your project directory, which will trigger the first-time setup flow upon refreshing the page.
  • Alternatively, run the setup script:
    python setup.py

If you are not the administrator:

Please contact the system administrator of this Origin instance to reset your password or assign new privileges.
